# AI Customization Instructions

이 저장소를 사용자가 "내 링크트리로 바꿔줘"라고 주면 바로 코드를 고치기 전에 아래 질문을 먼저 하세요. 답을 받으면 `src/config/linktree.ts`와 `src/config/theme.ts`만 우선 수정하세요.

## 먼저 물어볼 질문

1. 표시 이름은 무엇인가요? 예: DoRms 교사 OOO
2. 첫 화면 소개 문구에 들어갈 한 줄 설명은 무엇인가요?
3. `도름스 커뮤니티 나의 활동` 탭에 넣을 DoRms 활동 링크는 무엇인가요? 최소 1개, 가능하면 3개를 받으세요.
4. 외부 링크는 무엇인가요? 네이버 블로그, 인스타그램, 카카오톡, 자료 모음, 수업 앱, 강의 자료처럼 묶어 받을 수 있습니다.
5. 탭 안쪽 링크가 필요한 묶음은 무엇인가요? 각 묶음마다 링크 이름, 설명, URL을 받으세요.
6. 색감은 어떤 쪽이 좋나요? 따뜻한 크림, 차분한 파스텔, 밝은 교실톤, 진한 대비 중 하나로 물어보세요.
7. 아이콘은 어떤 것을 쓸까요? `public/icons`에 있는 SVG 이름 중에서 고르게 하세요.

## 고정 규칙

- 첫 번째 카드는 반드시 `도름스 커뮤니티 나의 활동`입니다.
- 첫 번째 카드의 대표 이미지는 `/assets/dorms-community.png`를 유지합니다.
- DoRms 팀 제품별 이미지나 서비스 내부 이미지는 추가하지 않습니다.
- DoRms 본 서비스의 코드, API, 인증, 환경변수, 운영 URL, 비공개 자산을 가져오지 않습니다.
- 사용자가 준 링크가 비공개이거나 개인정보가 들어간 URL이면 공개 저장소에 올려도 되는지 다시 확인합니다.
- 화면 문구는 쉬운 한국어로 씁니다. 코드와 식별자는 영어로 둡니다.
- 긴 줄표 문자는 쓰지 않습니다.

## 수정 위치

- 링크와 문구: `src/config/linktree.ts`
- 색상: `src/config/theme.ts`
- 아이콘 파일: `public/icons/*.svg`
- DoRms 커뮤니티 이미지: `public/assets/dorms-community.png`

## 검증

수정 후 아래를 실행하세요.

```bash
npm run build
```

가능하면 데스크톱과 모바일 폭에서 다음을 확인하세요.

- 첫 진입에서 스파이럴 화면이 보이는지
- `모든 활동 구경하기`를 누르면 링크 목록으로 넘어가는지
- 첫 번째 탭이 `도름스 커뮤니티 나의 활동`인지
- 탭 안 링크가 많을 때 내부 스크롤바가 기본 윈도우 모양이 아니라 템플릿 색으로 보이는지
